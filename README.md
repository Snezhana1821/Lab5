# Note App - Лабораторная работа по Scrum

## Описание проекта
Note App - это веб-приложение для управления заметками. Позволяет создавать заметки с метками, изменять их статусы и фильтровать по меткам и статусам.

**Основные возможности:**
* **Создание, просмотр, редактирование и удаление заметок**
* **Добавление меток (тегов) к заметкам**
* **Статусы заметок:** "Активные", "Выполненные", "В архиве"
* **Фильтрация заметок по меткам и статусам**
* **Поиск по тексту заметок**

**Технологии:**
* **Backend:** Python/Flask
* **Frontend:** JavaScript
* **Хранение данных:** JSON
* **Управление задачами:** GitHub Issues
